# Effective Go - Skill Instructions

## Focused Area

Core Go best practices and idioms from the official [Effective Go](https://go.dev/doc/effective_go) guide.

## Content Strategy

This skill provides quick reference for Go conventions:

- **Formatting**: gofmt usage and rules
- **Naming**: Package, variable, function naming
- **Error Handling**: Checking and wrapping errors
- **Interfaces**: Small interfaces and composition
- **Concurrency**: Channels vs mutexes
- **Documentation**: Godoc conventions

## Language Choice

**English** - The source material is in English and the concepts are technical.

## Update Guidelines

1. Keep SKILL.md as a concise index with links to references/
2. Add new reference files for additional Effective Go topics
3. Ensure all examples are idiomatic Go code
4. Link to official Go documentation as source
