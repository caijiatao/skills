# Golang Code Review - Skill Instructions

## Focused Area

Comprehensive code review based on "Golang Craftsmanship" (Golang 工匠).

## Content Strategy

This skill provides actionable code review guidance across five dimensions:

1. **命名规范**: Package, variable, function, constant, interface naming
2. **结构体设计**: Field ordering, embedding, constructors
3. **函数设计**: Single responsibility, parameter design
4. **接口设计**: Small interfaces, composition
5. **错误处理**: Error creation, wrapping, checking

## Language Choice

**Chinese (zh)** - Based on Chinese translation of Uber Go Style Guide.

## Update Guidelines

1. Maintain Chinese language for accessibility
2. Keep review checklist format
3. Add before/after code examples
4. Link to both Chinese and English resources
