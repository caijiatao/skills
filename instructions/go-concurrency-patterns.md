# Go Concurrency Patterns - Skill Instructions

## Focused Area

Production-ready concurrency patterns in Go.

## Content Strategy

This skill covers practical concurrency patterns:

- **Core Primitives**: Goroutines, channels, select, sync
- **Patterns**: Worker pools, pipelines, fan-out/fan-in
- **Best Practices**: Graceful shutdown, error groups
- **Debugging**: Race detection

## Language Choice

**English** - Technical concurrency concepts and patterns.

## Update Guidelines

1. Focus on production-proven patterns
2. Include complete, working examples
3. Cover both basic and advanced patterns
4. Emphasize common pitfalls and how to avoid them
